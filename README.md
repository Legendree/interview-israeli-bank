# Frontend Dev Task

Built with the following 3rd party packages:

- Axios
- Redux
- React-Redux
- React-Router-Dom

Live example can be viewed at: https://interview-bank.vercel.app/

## Installation

Clone this repo to your desktop

```
git clone https://github.com/Legendree/interview-israeli-bank.git
```

Run `yarn` to fetch necessary dependencies

Run `yarn start` to start the development server and view the application.

**Please ensure the app running on Port 3000, as the app requires Spotify authentication via URI Redirects with the same port on Spotifies End**
